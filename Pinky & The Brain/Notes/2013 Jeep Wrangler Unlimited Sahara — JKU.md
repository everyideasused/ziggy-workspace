### Pentastar Engine Notes

- The Pentastar V6 replaced the underpowered 3.8L V6 starting in 2012 model year
- The 2013 model uses updated cylinder heads (part number RL141353AC) with hardened valve guides and seats starting around Julian date 2062 (July 2012 production)
- Earlier 2013 builds (pre-January 2013) may still have original head castings
- Exhaust manifolds are integrated into cylinder heads — exhaust leaks uncommon
- Uses variable valve timing (VVT) on both intake and exhaust camshafts
- Oil filter housing is a plastic/aluminum assembly bolted to engine block (NOT traditional spin-on) — known failure point

## Reading Diagnostic Trouble Codes (No Scanner Required)

The JK Wrangler has a built-in DTC reader accessible through the ignition key:

1. Insert key into ignition
2. Cycle: ON → OFF → ON → OFF → ON (leave in ON position)
3. Do this quickly (within ~5 seconds), pressing the key in so it doesn't catch at ACC
4. The dash lights up fully, odometer shows dashes `------`
5. Stored codes scroll across the odometer display (letter + 4 digits)
6. If no codes: displays `-done-`

**Limitations:** This method is read-only. You cannot clear codes without an OBD-II scan tool. Some 2013 models may not display the actual code characters — only dashes. In that case, use JScan (Bluetooth OBD-II app), a basic code reader, or dealer scan tool.

### DTC Structure

| Position | Meaning |
|---|---|
| 1st character (letter) | P = Powertrain, B = Body, C = Chassis, U = Network |
| 2nd character (digit) | 0 = SAE standard, 1 = Manufacturer-specific |
| 3rd character (digit) | Subsystem (1=Fuel/Air, 2=Injector, 3=Ignition/Misfire, 4=Aux Emissions, 5=Speed/Idle, 6=Computer Output, 7-8=Transmission) |
| 4th–5th characters | Specific fault (00–99) |

## Common DTC Codes & Troubleshooting Trees

### Engine Misfire Codes

| Code | Description | Common Cause on 3.6L |
|---|---|---|
| P0300 | Random/Multiple Cylinder Misfire | Fouled plugs, coil pack, vacuum leak, low fuel pressure, cylinder head issue |
| P0301 | Cylinder 1 Misfire | Bad coil, plug, injector |
| P0302 | Cylinder 2 Misfire | **Left cylinder head valve seat failure** (early builds) |
| P0303 | Cylinder 3 Misfire | Bad coil, plug, injector |
| P0304 | Cylinder 4 Misfire | **Left cylinder head valve seat failure** (early builds) |
| P0305 | Cylinder 5 Misfire | Bad coil, plug, injector |
| P0306 | Cylinder 6 Misfire | **Left cylinder head valve seat failure** (early builds) |

**Troubleshooting Tree — Misfire:**
1. Pull codes → identify which cylinder(s)
2. If P0302/P0304/P0306 (left bank only) → check left cylinder head date stamp. If pre-2062, suspect valve seat wear → leak-down test → head replacement under TSB D-14-12 (extended warranty)
3. If single cylinder → swap coil pack with a known-good cylinder. If misfire follows, coil is bad. If misfire stays, check plug → injector → compression
4. If random P0300 → check for vacuum leaks, fuel pressure, spark plugs

### Oxygen Sensor Codes

| Code | Description |
|---|---|
| P0131 | O2 Sensor B1S1 Low Voltage |
| P0132 | O2 Sensor B1S1 High Voltage |
| P0133 | O2 Sensor B1S1 Slow Response |
| P0135 | O2 Sensor B1S1 Heater Circuit |
| P0137 | O2 Sensor B1S2 Low Voltage |
| P0138 | O2 Sensor B1S2 High Voltage |
| P0141 | O2 Sensor B1S2 Heater Circuit |
| P0151 | O2 Sensor B2S1 Low Voltage |
| P0152 | O2 Sensor B2S1 High Voltage |
| P0153 | O2 Sensor B2S1 Slow Response |
| P0157 | O2 Sensor B2S2 Low Voltage |
| P0158 | O2 Sensor B2S2 High Voltage |
| P0160 | O2 Sensor B2S2 No Activity |
| P1128 | Closed Loop Not Achieved B1 |
| P1129 | Closed Loop Not Achieved B2 |

**Critical Note:** The 3.6L Pentastar uses a **ground-biased O2 sensor** design. Aftermarket sensors (especially non-NTK/Denso) frequently cause false codes and closed-loop failures. **Always use OEM Mopar or NTK sensors** on this engine.

**Troubleshooting Tree — O2 Codes:**
1. Check PCV valve (passenger side rear of valve cover) — oil leaking onto sensor causes false readings
2. Verify sensor wiring harness is not melted against exhaust manifold
3. If replacing, use OEM or NTK only
4. After replacement, clear codes and drive through 2–3 complete drive cycles

## Troubleshooting Trees

### Engine Won't Start

| Symptom | Probable Cause | Diagnosis |
|---|---|---|
| Cranks but won't fire | No fuel (pump failure, relay, fuse F18), no spark (CKP sensor, coil failure), WCM/SKREEM key transponder failure (B2229), immobilizer issue | Listen for fuel pump prime (2-sec buzz when key turns to ON). Check spark by pulling a coil/plug. If no prime → check fuel pump relay (K4) and fuse F18. If prime OK but no start → check CKP sensor and WCM |
| No crank, no click | Dead battery, bad starter, starter relay (K8), blown fuse F41, neutral safety switch (auto) or clutch pedal switch (manual), ignition switch | Test battery voltage >12.4V. Check starter relay. Try starting in Neutral vs Park |
| Clicks but no crank | Weak battery, poor battery connection, bad starter solenoid | Load-test battery. Check/clean terminals. If battery good, likely starter motor |

### Overheating

| Symptom | Probable Cause | Diagnosis |
|---|---|---|
| Overheating at idle/low speed | Cooling fan failure (electric fan or clutch fan), blocked radiator, low coolant, radiator cap not holding pressure | Check fan operation. Verify coolant level in radiator (not just overflow). Pressure-test cooling system |
| Overheating at highway speed | Restricted radiator airflow (mud/debris), partially plugged radiator internally, water pump impeller failure, stuck thermostat | Check radiator for blockage. If coolant looks muddy/rusty, likely internal radiator plugging |
| Rapid overheating after warm-up | Stuck-closed thermostat, head gasket leak (exhaust gases in coolant), water pump failure | After cooling, check thermostat by boiling in water. Test for exhaust gases in coolant |

### Rough Idle / Stalling

| Symptom | Probable Cause | Diagnosis |
|---|---|---|
| Rough idle, runs OK under load | Vacuum leak (intake manifold gaskets, PCV valve, brake booster line), dirty throttle body, dirty/bad idle air control (if equipped) | Smoke test for vacuum leaks. Clean throttle body. Check PCV valve |
| Stalls at stops | Same as above, plus possible torque converter clutch stuck engaged (auto transmission) | Check for vacuum leaks first. If auto trans, does it feel like manual trans when stopping? TCC stuck on |
| Surging idle, hunting RPM | Vacuum leak, failing MAP sensor, dirty throttle body, failing IAC (if equipped) | Check MAP sensor with scan tool. Smoke test for vacuum leaks |

### Poor Performance / Lack of Power

| Symptom | Probable Cause | Diagnosis |
|---|---|---|
| Sluggish acceleration, no codes | Restricted exhaust (catalytic converter), clogged fuel filter, weak fuel pump, restricted air filter, dragging brakes | Check exhaust backpressure. Test fuel pressure. Check brake calipers for drag |
| Power loss under load only | Restricted exhaust, weak fuel pump, bad MAF/MAP sensor | Monitor fuel pressure under load. Check exhaust backpressure |
| Power loss + misfire codes | Ignition coils, spark plugs, injector(s), low compression | Pull plugs and inspect. Test coils. Compression test |

### Transmission Issues (Automatic)

| Symptom | Probable Cause | Diagnosis |
|---|---|---|
| Delayed engagement (D or R) | Low ATF, worn pump, valve body issue, TCM programming | Check ATF level (warm, in Park, level ground). If OK, likely valve body or pump wear |
| Harsh shifting | Low ATF, dirty/old fluid, valve body wear, solenoid failure, TCM adaptive reset needed | Check fluid. If dark/burned → service. If OK, reset TCM adaptive memory |
| Slipping between gears | Low fluid, worn clutches, valve body issue | Check fluid. If slipping continues with proper level → internal wear (rebuild likely) |
| Won't shift out of 1st gear (limp mode) | TCM detected fault, usually electrical (solenoid, sensor, wiring) | Scan for codes. Could be simple electrical or internal mechanical |

## Service Procedures

### Engine Oil Change — 3.6L Pentastar

**Interval:** 6,000–8,000 miles (synthetic) / per Oil Life Monitor

**Tools:**
- 13mm socket + ratchet (drain plug)
- 24mm deep socket + ratchet + 4" extension (filter cap)
- Torque wrench (ft-lb range)
- Oil filter wrench (backup for stuck cap)

**Procedure:**
1. Warm engine to operating temp (drives contaminants into suspension)
2. Remove oil fill cap (allows air in, faster draining)
3. Position drain pan under engine
4. Remove 13mm drain plug (back of oil pan, driver side) using socket
5. Allow to drain 10–15 minutes (full drain)
6. Clean drain plug, apply new crush washer (M14 crush washer — replace every time)
7. Reinstall drain plug: torque to **25 ft-lb**
8. Locate oil filter housing (passenger side of engine, front/center, gray/black plastic cap with hex top)
9. Remove using 24mm deep socket + extension (socket goes on hex protrusion on cap's top)
10. Remove old filter from cap (pry off — watch orientation)
11. Replace O-ring on cap (large black rubber ring). Lightly coat with fresh oil
12. Install new filter onto cap snaps into place, correct orientation)
13. Reinstall cap/filter assembly: hand-tighten, then torque to **18 ft-lb** (or hand-tight + ¼ turn)
14. Add 6 quarts of 5W-20 synthetic oil ( Mobil 1, Pennzoil Platinum, Mopar)
15. Reinstall oil fill cap
16. Start engine, idle for 30 seconds (oil pressure light goes off)
17. Shut off, wait 5 minutes, check dipstick (should be at Full mark)
18. Check under vehicle for leaks at drain plug and filter housing
19. Reset Oil Life Monitor: turn key to ON (don't start), press gas pedal to floor 3 times within 10 seconds

**OEM Oil Spec:** 5W-20 API SN/SP, synthetic recommended (Pennzoil Platinum, Mobil 1, Mopar)
**Capacity:** 6 quarts with filter change
**Filter:** Mopar 68191349AC (uses cartridge, not spin-on)

### Spark Plug Replacement — 3.6L Pentastar

**Interval:** 100,000 miles (iridium plugs)

**Tools:**
- 5/8" thin-wall spark plug socket with rubber insert (holds plug)
- 3/8" extensions: 3", 6"
- 3/8" ratchet
- Torque wrench
- Dielectric grease (for coil boots)
- Anti-seize (copper-based, for plug threads)

**Procedure — Bank 1 (Passenger Side — cylinders 1-3-5):**
1. Remove engine cover (if equipped) — pulls straight up
2. Locate coil packs on passenger side valve cover (3 of them, in a row)
3. Disconnect electrical connector from each coil (squeeze tab, pull)
4. Remove 10mm bolt holding coil to valve cover
5. Pull coil straight up off plug — may need gentle wiggle
6. Use 5/8" socket + extension to remove spark plug (counterclockwise)
7. Gap new plugs to 0.043" if not pre-gapped. Apply small amount of copper anti-seize to threads only
8. Hand-thread new plug into head — do NOT cross-thread
9. Torque to **20 ft-lb** (lightly snug — iridium plugs don't need much)
10. Apply dielectric grease inside coil boot (prevents misfires from moisture/corrosion)
11. Reinstall coil onto plug, bolt down (hand-tight + ¼ turn)
12. Reconnect electrical connector (click when seated)

**Procedure — Bank 2 (Driver Side — cylinders 2-4-6):**

**⚠️ Difficult access — intake manifold must be loosened**

1. Remove decorative engine cover
2. Remove windshield wiper arms (15mm nut under cap, wiper arm puller helps)
3. Remove plastic cowl cover (screws along base of windshield)
4. Remove intake manifold support bracket (if equipped)
5. Loosen (don't remove) intake manifold bolts enough to lift manifold ~2" for coil access
6. Support manifold with wood block or similar (don't stress hoses/wires)
7. Now access Bank 2 coil packs and remove/replace plugs as above
8. Reassemble in reverse order:
   - Torque intake manifold bolts: **105 in-lb** (inch-pounds, NOT ft-lb — critical!)
   - Torque wiper arm nuts: **21 ft-lb**

**Plug Spec:** NGK ILKR9G8 (iridium), Champion RE14MCC4 (copper), Mopar SP149125AF
**Gap:** 0.043" (1.1mm)
**Torque:** 20 ft-lb

### Tire Rotation

**Interval:** 6,000–8,000 miles

**Rotation Pattern:**
- Rear-wheel drive pattern (solid rear axle):
  - Front tires → Rear (same side)
  - Rear tires → Front (same side)
- **DO NOT cross-rotate** on JK with limited slip or Trac-Lok differential

**Procedure:**
1. Vehicle on level ground, transmission in Park, parking brake on
2. Loosen lug nuts BREAK TORQUE ONLY while tire on ground (½ turn loose)
3. Jack up vehicle using approved jacking points (frame rails, NOT axle tubes for rotation)
4. Support on jack stands — never work under vehicle supported by jack alone
5. Remove and reposition tires per pattern
6. Hand-thread all lug nuts (prevents cross-threading)
7. Lower vehicle until tires just touch ground
8. Torque lug nuts in STAR PATTERN to **95 ft-lb**
9. Final torque check after 50–100 miles

**Lug Nut Torque:** 95 ft-lb
**Pattern:** Star pattern, NOT clockwise

### Brake Pad Replacement — Front Disc Brakes

**Tools:**
- 13mm socket/ratchet (caliper bolts)
- 18mm socket/ratchet (caliper bracket bolts)
- C-clamp or caliper piston tool (to compress piston)
- Torque wrench
- Brake cleaner
- Anti-seize (for slide pins)

**Procedure:**
1. Loosen lug nuts (break torque only)
2. Jack up vehicle, support on stands
3. Remove wheel
4. Remove lower caliper bolt (13mm) — swing caliper up out of way (don't let hang by hose)
5. Remove old brake pads
6. Inspect rotors — measure thickness, check for scoring/grooves. Minimum thickness: 0.984" front
7. Compress piston using C-clamp or piston tool (old pad over piston, push slowly)
8. Install new pads (may need to push pistons further if pads are thicker)
9. Apply thin coat of brake lubricant to back of pads (squeal prevention)
10. Swing caliper down over new pads
11. Install caliper bolt: torque to **26 ft-lb**
12. **Check brake fluid level** — compressing pistons raises fluid. If reservoir full, remove some with turkey baster to prevent overflow
13. Reinstall wheel, torque lugs to **95 ft-lb**
14. Repeat other side
15. Before driving: pump brake pedal 10–15 times until firm
16. Bed-in procedure: 5 moderate stops from 40→10 mph with cooling between

**Torque Specs:**
- Caliper bolts: 26 ft-lb
- Caliper bracket bolts: 85 ft-lb
- Lug nuts: 95 ft-lb

**Pad Spec:** Semi-metallic or ceramic compatible with factory calipers
