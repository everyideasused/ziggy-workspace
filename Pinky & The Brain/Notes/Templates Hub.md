# 📋 Templates Hub
## When to Use Each Template

| Template | Use When... | Creates Type |
|----------|------------|--------------|
| **Daily Note** | Auto-created each day via Calendar plugin | Journal entry |
| **Weekly Review** | Friday or weekend — weekly processing ritual | Review note |
| **Quick Capture** | Need to jot something down fast, categorize later | `inbox` note |
| **Project** | Starting any personal project (non-work) | `project` |
| **Work Client Project** | Onboarding a new client in your PMO | `project` (work/client) |
| **Work Site Project** | Adding an individual construction project/site | `work-project` |
| **Area Hub** | Creating a new life area hub (one-time setup) | `area-hub` |
| **Meeting** | Before or after any meeting | `meeting` |
| **Person** | First meaningful interaction with someone new | `person` |
| **Organization** | New company, agency, or group worth tracking | `organization` |
| **Item** | New purchase, subscription, or trackable asset | `item` |
| **Account** | New financial account (bank, card, loan, investment) | `account` |
| **Bill** | New recurring payment or bill | `bill` |
| **Monthly Finance Review** | First of each month — full financial check-in | `monthly-finance` |
| **Goal** | Setting a new measurable goal | `goal` |
| **Workout Day A** | Upper Push day (pecs, shoulders, triceps) | `workout` |
| **Workout Day B1** | Lower day — glute/quad focus | `workout` |
| **Workout Day B2** | Lower day — posterior chain/glute focus | `workout` |
| **Workout Day C** | Upper Pull + Arms day (back, biceps) | `workout` |
| **Fitness Assessment** | Every 4 weeks — mesocycle review | `assessment` |
| **Habit** | New recurring habit to track daily/weekly | `habit` |
| **Book** | Start reading or add to reading list | `book` |
| **Recipe** | Save a recipe with ingredients and macros | `recipe` |
| **Resource** | Saving reference material for later | `resource` |
| Session State | End-of-session summary for Ziggy continuity | system |

### How to Apply a Template
1. Create a new note (Cmd+N)
2. Open command palette (Cmd+P)
3. Type "Templater: Insert template"
4. Select the template you want
5. Fill in the prompts that appear
