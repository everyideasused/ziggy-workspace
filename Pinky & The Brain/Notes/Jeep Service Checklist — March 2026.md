## 🚗 Saturday: Grand Cherokee Service

### Pre-Flight
- [ ] **Warm engine** (5 minutes)
- [ ] **Park on level surface** (engage parking brake, chock wheels)
- [ ] **Let engine cool** (wait 10 min before starting)

### Oil Change (30-45 min)
- [ ] Remove engine cover
- [ ] Remove oil cap, pull dipstick up
- [ ] **Loosen oil filter cap FIRST** (24mm)
- [ ] Position drain pan
- [ ] Remove oil drain bolt (13mm) — ⚠️ hot oil
- [ ] Drain 5+ minutes
- [ ] Replace drain bolt (**20 lb-ft torque**)
- [ ] Remove old oil filter + o-ring
- [ ] Lubricate new o-ring
- [ ] Install new filter (Mopar 68191349AA or K&N PS-7026)
- [ ] Tighten filter cap (**hand-tight + 1/4 turn**)
- [ ] Pour 5-5.5qt oil (Synthetic 5W-20)
- [ ] Run engine 2 min, shut off, wait 5 min
- [ ] Check oil level (top off to "SAFE")
- [ ] Replace engine cover
- [ ] **Reset oil change indicator** (gas pedal 3x trick)

### Coolant Flush + Thermostat (1-1.5 hrs)
- [ ] **Engine COLD** — verify before starting
- [ ] Position drain pan under radiator
- [ ] Open radiator petcock (driver side, bottom)
- [ ] Remove radiator cap (speeds draining)
- [ ] Drain coolant completely
- [ ] **Disconnect upper radiator hose** (thermostat housing)
- [ ] **Remove thermostat housing bolts**
- [ ] **Clean mating surface** (scrape old gasket)
- [ ] **Install new thermostat** (Mopar 68253514AA — 203°F)
- [ ] **Torque housing bolts** (verify spec — 10-15 lb-ft typical)
- [ ] **Reconnect upper radiator hose** (tighten clamp)
- [ ] **Flush with distilled water** (2-3 cycles)
- [ ] Close petcock
- [ ] **Mix coolant** (2 gal Mopar OAT purple + 2 gal distilled = 50/50)
- [ ] Fill radiator with 50/50 mix
- [ ] Fill overflow reservoir to MAX
- [ ] **Burp system** (engine running, cap off, heater HIGH, squeeze hose)
- [ ] Top off as air escapes
- [ ] Replace radiator cap (when no bubbles, at operating temp)
- [ ] **Test coolant** (-34°F freeze protection)
- [ ] Check for leaks (hoses, clamps, thermostat housing)

### Air Filters (15 min)
- [ ] **Engine air filter:** Open box, replace (Mopar 68257791AB), close
- [ ] **Cabin air filter:** Drop glove box, replace (Mopar 68071668AA), reinstall

### Final Checks
- [ ] Start engine, listen for unusual sounds
- [ ] Check for leaks (oil drain bolt, filter cap, thermostat housing, radiator)
- [ ] **Test drive** (5-10 minutes)
- [ ] Re-check fluid levels (oil, coolant)
- [ ] **Update maintenance log** in vault

### Clean Up
- [ ] Drain old oil into disposal containers
- [ ] Drain old coolant into disposal containers
- [ ] Wipe down tools
- [ ] Organize workspace for Sunday
