# 05 - Procurement & Bidding

> **Phase 5 of 10** | Predecessor: [[03 - Design — SD, DD, CD]] | Successor: [[06 - Preconstruction]]

---

## Purpose

Select and contract the construction team (GC, CM, or trade contractors) and procure long-lead materials. The approach depends entirely on the chosen [[Project Delivery Methods|project delivery method]].

---

## Procurement by Delivery Method

### Design-Bid-Build (DBB)
```
CDs Complete → Bid Documents Prepared → Advertise/Invite → 
Pre-Bid Meeting → Bidding Period (2-4 weeks) → Bid Opening → 
Bid Evaluation → Award → Contract Execution
```
- **Bid Documents:** Drawings, specs, front-end (instructions to bidders, bid form, contract form, general/special conditions)
- **Evaluation:** Lowest responsible and responsive bidder (public); best value (private)
- **Contract:** Typically AIA A101 (Stipulated Sum)

### Construction Manager at Risk (CMAR)
```
CM Selected During Design → GMP Negotiation at Late DD/Early CD → 
Trade Packages Bid by CM → GMP Established → Contract Execution
```
- CM is selected on qualifications (RFQ/RFP process), not low bid
- CM provides preconstruction services during design
- GMP set at 60-90% design completion
- CM bids trade packages to subcontractors (open-book)
- **Contract:** Typically AIA A102 (Cost Plus with GMP)

### Design-Build (DB)
```
Owner's Criteria Developed → RFQ for DB Teams → Shortlist → 
RFP with Criteria Package → Proposals Received → Selection → Contract
```
- Single entity responsible for both design and construction
- Selection based on qualifications, design approach, and price
- **Contract:** Typically AIA A141 (Design-Build)

### Integrated Project Delivery (IPD)
```
Key Parties Selected → Multi-Party Agreement → Validation Phase →
Target Cost Established → Proceed to Design/Build
```
- Owner, architect, and contractor selected simultaneously
- Single multi-party contract (AIA C191)
- Shared risk/reward pool

## GC/CM Selection Process

### Request for Qualifications (RFQ)
Evaluates:
- Firm experience with similar project types
- Key personnel qualifications and availability
- Current workload and capacity
- Financial strength and bonding capacity
- Safety record (EMR, OSHA citations)
- References from similar projects
- Local experience and AHJ relationships

### Request for Proposal (RFP)
Adds to RFQ:
- Approach to the specific project
- Preliminary schedule
- Preliminary pricing (fee, general conditions, contingency)
- Proposed subcontractor outreach plan
- Value engineering ideas
- BIM/technology approach

### Interview/Presentation
- Key personnel must attend (PM, Superintendent)
- Chemistry and communication assessment
- Scenario-based questions

## Bidding Process (Subcontractor Level)

Whether the GC/CM bids trade packages:

### Trade Packages (Typical)
| Package | Scope |
|---------|-------|
| Sitework/Earthwork | Grading, utilities, paving |
| Concrete | Foundations, slabs, tilt-up |
| Structural Steel | Framing, connections, erection |
| Framing (wood/metal stud) | Interior and exterior framing |
| Roofing | Membrane, flashing, insulation |
| Exterior Envelope | Curtainwall, storefront, cladding |
| Drywall/Plaster | Interior finishes |
| Flooring | Tile, carpet, LVT, polished concrete |
| Painting | Interior and exterior |
| Mechanical (HVAC) | Equipment, ductwork, controls |
| Plumbing | Piping, fixtures, water heaters |
| Electrical | Power, lighting, low-voltage |
| Fire Protection | Sprinklers, standpipes, alarms |
| Elevator | Vertical transportation |
| Specialties | Signage, toilet accessories, food service equipment |

### Bid Evaluation Criteria
- Base bid amount
- Alternates (add/deduct pricing for optional scope items)
- Unit prices (for anticipated variable quantities)
- Schedule compliance
- Scope completeness (are there exclusions?)
- Qualifications and exceptions
- Subcontractor's financial strength
- Past performance

## Long-Lead Item Procurement

Items requiring early ordering due to extended fabrication/delivery times:

| Item | Typical Lead Time |
|------|------------------|
| Structural steel | 8-16 weeks |
| Switchgear/electrical panels | 12-24 weeks |
| Generators | 16-30 weeks |
| Elevators | 16-30 weeks |
| Custom curtainwall | 12-20 weeks |
| Rooftop HVAC units | 8-16 weeks |
| Custom millwork | 8-14 weeks |
| Specialty tile/stone | 8-16 weeks |

**Strategy:** Identify long-lead items during DD, begin procurement during CD, award early packages before full contract execution if needed.

## Contract Execution Checklist

- [ ] Scope of work clearly defined
- [ ] Contract form selected and negotiated (see [[Contract Types & Structures]])
- [ ] Insurance requirements specified and certificates received
- [ ] Bonding requirements met (payment bond, performance bond)
- [ ] Schedule of values submitted and approved (see [[Financial Management & Billing]])
- [ ] Retainage terms established
- [ ] Change order process defined
- [ ] Dispute resolution mechanism agreed
- [ ] Liquidated damages (if applicable) established
- [ ] Owner-furnished items identified
- [ ] Allowances defined and budgeted
- [ ] Permits and fees responsibility assigned
- [ ] Warranty terms specified

## Duration Benchmarks

| Activity | Duration |
|---------|----------|
| RFQ/RFP for GC/CM | 3-6 weeks |
| Subcontractor bidding period | 2-4 weeks |
| Bid evaluation and negotiation | 1-2 weeks |
| Contract negotiation and execution | 2-4 weeks |
| **Total phase** | **4-12 weeks** |

## Transition to Next Phase

**Gate criteria to advance to [[06 - Preconstruction]]:**
1. GC/CM contract fully executed
2. Key subcontractor contracts in progress or awarded
3. Long-lead items identified and ordered (or purchase orders pending)
4. Insurance and bonding in place
5. Schedule of values submitted
