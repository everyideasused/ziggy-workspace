# 03 - Design — SD, DD, CD

> **Phase 3 of 10** | Predecessor: [[02 - Feasibility & Due Diligence]] | Successor: [[04 - Entitlements & Permitting]], [[05 - Procurement & Bidding]]

---

## Purpose

Transform the owner's program requirements into buildable construction documents through progressive design refinement. The AIA divides this into three sub-phases: Schematic Design (SD), Design Development (DD), and Construction Documents (CD).

---

## Sub-Phase 3A: Schematic Design (SD) — ~20% Design Completion

### Purpose
Establish the overall design concept, spatial relationships, and project scope at a high level.

### Key Activities
- Translate owner's program (space requirements, adjacencies, brand standards) into conceptual plans
- Develop site plan showing building placement, parking, circulation, landscaping
- Create preliminary floor plans showing room layouts and spatial relationships
- Produce exterior elevation concepts (massing, materials palette, character)
- Conduct preliminary code analysis (occupancy classification, egress, accessibility)
- Coordinate with civil engineer on site constraints (setbacks, stormwater, grading)
- Begin sustainability/LEED strategy if applicable
- Develop preliminary construction cost estimate (±20-30%)
- Multiple design iterations based on owner feedback

### SD Deliverables
- [ ] Site plan (conceptual)
- [ ] Floor plans (conceptual, showing major spaces)
- [ ] Building sections (major)
- [ ] Exterior elevations (conceptual)
- [ ] Preliminary code analysis
- [ ] Area calculations
- [ ] Preliminary construction cost estimate
- [ ] SD presentation package
- [ ] **Owner approval to proceed to DD**

### Who's Involved
- Architect (lead), Owner, Civil Engineer, possibly Landscape Architect
- Specialty consultants join at end of SD or beginning of DD
- If CMAR delivery: Construction Manager begins providing cost input

---

## Sub-Phase 3B: Design Development (DD) — ~70% Design Completion

### Purpose
Refine the approved schematic design into a detailed, coordinated design with systems integration, material selections, and engineering.

### Key Activities
- Refine floor plans with detailed room layouts, dimensions, and finishes
- Develop detailed building sections and wall sections
- Select and specify major materials (exterior cladding, roofing, flooring, ceiling systems)
- Coordinate structural system design (steel, concrete, wood framing, etc.)
- Design MEP systems (HVAC, plumbing, electrical, fire protection)
- Develop reflected ceiling plans
- Finalize interior design selections and finish schedules
- Code compliance review with all disciplines
- Update construction cost estimate (±10-15%)
- Conduct constructability review (especially with CMAR/DB)
- Begin coordination between engineering disciplines (clash detection if using BIM)
- Address ADA/accessibility requirements in detail

### DD Deliverables
- [ ] Refined floor plans with dimensions
- [ ] Building sections and wall sections
- [ ] Exterior elevations with materials called out
- [ ] Reflected ceiling plans
- [ ] Structural design drawings
- [ ] MEP schematic design drawings
- [ ] Preliminary finish schedule
- [ ] Updated cost estimate
- [ ] Outline specifications
- [ ] **Owner approval to proceed to CD**

### Who's Involved
- Architect, Structural Engineer, MEP Engineers, Civil Engineer
- Interior Designer, Landscape Architect
- Specialty consultants: Acoustical, AV, Food Service, Low-Voltage, Security
- Cost Estimator, Construction Manager (if CMAR)
- Owner (review and approve)

---

## Sub-Phase 3C: Construction Documents (CD) — 100% Design Completion

### Purpose
Produce the complete, detailed, coordinated set of drawings and specifications that will be submitted for permitting, used for bidding, and guide construction.

### Key Activities
- Complete all architectural drawing sheets (plans, elevations, sections, details, schedules)
- Complete all engineering drawing sheets (structural, MEP, civil, landscape)
- Write Project Manual (specifications organized by CSI MasterFormat divisions)
- Finalize door, window, finish, and equipment schedules
- Complete code compliance documentation
- Perform quality control review (interdisciplinary coordination check)
- Final cost estimate based on complete documents (±5-10%)
- Prepare bid documents (if DBB delivery)
- Address all outstanding design decisions
- Life safety and fire protection plan development

### CD Deliverables
- [ ] Complete architectural drawing set
- [ ] Complete structural drawing set
- [ ] Complete MEP drawing set (mechanical, electrical, plumbing, fire protection)
- [ ] Civil/site drawing set
- [ ] Landscape drawing set
- [ ] Project Manual (specifications, Divisions 00-49 per CSI format)
- [ ] Final cost estimate
- [ ] Bid documents (front-end, bid forms, instructions to bidders) — if DBB
- [ ] Permit application package
- [ ] **Owner approval of final documents**

## Design Phase Completion Matrix

| Element | SD | DD | CD |
|---------|----|----|-----|
| Floor plans | Bubble diagrams, major rooms | Dimensioned, partitions shown | Fully detailed with notes |
| Elevations | Massing, character | Materials, fenestration | Full dimensions, details referenced |
| Structural | Not started | System selected, preliminary sizing | Complete design with connections |
| HVAC | Not started | System selected, equipment sized | Ductwork/piping routed, schedules |
| Electrical | Not started | Panel sizing, major circuits | Complete circuiting, lighting layout |
| Plumbing | Not started | Fixture counts, riser diagrams | Full piping layout, schedules |
| Specifications | None | Outline specs | Full Project Manual |
| Cost estimate | ±30% | ±15% | ±5-10% |

---

## Critical Coordination Points

### Between Disciplines
- Structural vs. MEP: Beam/duct routing, penetrations, equipment loads
- Architectural vs. Structural: Floor-to-floor height, column grid, lateral system
- Fire Protection vs. All: Sprinkler head locations, fire-rated assemblies, firestopping
- Civil vs. Architectural: Finished floor elevation, building pad, utility connections

### Between Design and Other Phases
- **Permitting:** CD set must be complete enough for permit submission; some AHJs accept 90% CDs
- **Procurement:** GMP pricing typically occurs at late DD or early CD (60-90% design)
- **Long-lead items:** Identify early in DD for procurement during CD phase

---

## Owner's Design Responsibilities

The owner/client must provide timely:
- Program requirements and space needs
- Brand standards and design guidelines (retail)
- Equipment and FF&E specifications
- Review and approval at each phase gate (SD → DD → CD)
- Decision-making on value engineering options
- Budget authorization for scope changes

**Warning:** Owner delays in review/approval are the #1 cause of design schedule overruns.

## Duration Benchmarks

| Sub-Phase | Simple Renovation | Medium Commercial | Complex New Build |
|-----------|------------------|-------------------|-------------------|
| SD | 2-3 weeks | 4-6 weeks | 6-12 weeks |
| DD | 3-4 weeks | 6-8 weeks | 8-16 weeks |
| CD | 3-4 weeks | 6-10 weeks | 10-20 weeks |
| **Total** | **8-11 weeks** | **16-24 weeks** | **24-48 weeks** |

## Dependencies & Predecessors

| Dependency | Source | Impact if Missing |
|-----------|--------|-------------------|
| Survey/topo | Phase 2 | Civil and site design cannot begin |
| Geotech report | Phase 2 | Structural design has no foundation basis |
| Zoning analysis | Phase 2 | Design may violate zoning and require redesign |
| Owner program | Owner/Client | Architect cannot design without requirements |
| Existing conditions (reno) | Field survey | Design based on assumptions = costly RFIs |

---

## Common Risks

| Risk | Mitigation |
|------|------------|
| Owner scope creep during design | Establish formal change management process; price all adds |
| Discipline coordination failures | Regular coordination meetings; BIM clash detection |
| Design exceeds budget | Engage cost estimator at each sub-phase; VE workshops |
| Incomplete or inaccurate existing conditions | Invest in thorough as-built verification before design |
| Permitting requirements not anticipated | Engage AHJ early (pre-application meetings) |

---

## Transition to Next Phase

**Gate criteria to advance to [[04 - Entitlements & Permitting]] and [[05 - Procurement & Bidding]]:**
1. CD set is complete and owner-approved
2. Project Manual (specifications) is complete
3. Final cost estimate is within budget parameters
4. All design disciplines are coordinated
5. Permit application package is assembled
