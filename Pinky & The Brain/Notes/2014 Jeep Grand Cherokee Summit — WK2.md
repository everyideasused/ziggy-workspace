### Shared Pentastar Issues with JK

**All these JK Pentastar issues apply to WK2:**
- Left cylinder head valve seat (TSB D-14-12, 10yr/150k extended warranty)
- Pentastar tick (rocker arm roller bearing failure)
- Oil filter housing leak (plastic/aluminum assembly, O-ring failure)
- Cracked thermostat housing (plastic → consider aluminum)
- Timing chain tensioner wear (cold-start rattle)

### 2014-Specific Notes

- 2014 was first year of ZF 8-speed pairing with 3.6L
- Should have **updated cylinder heads** (RL141353AC with hardened valve seats)
- Variable displacement oil pump (2-stage) may be present — verify by checking oil pressure behavior

### WK2-Specific Chassis Codes (C-Codes)

| Code | Description | Common Cause |
|---|---|---|
| C1562 | Compressor Performance | Fuse F32 first, then compressor |
| C156E | Compressor Over-Temp | Leak causing extended run |
| C15A1 | System Pressure Low | Air spring leak, line leak |
| C15D9 | Air Spring Solenoid | Replace corner air spring |

| C2212 | In-Plant Mode | Run "Fill From Reservoir" routine |

### ⚠️ Cascading Electrical Fault Pattern

A battery testing at 12.2V or below can generate 10–20 simultaneous fault codes across PCM, ABS, EPAS, air suspension, UConnect, TPMS, ELSD. **Test battery under load BEFORE chasing any C/B/U code.**

## Service Procedures

### Engine Oil Change (Same as JK)
- 13mm drain plug @ 20 ft-lb
- 24mm filter cap @ 18 ft-lb
- 6 qt 5W-20 synthetic
- Mopar 68191349AA filter cartridge
- Oil Life Reset: Key ON → gas pedal 3x in 10 sec

### ZF 8-Speed Transmission Service
**Interval:** 80,000–100,000 miles (despite "lifetime" claim)

1. Warm transmission (drive 10+ miles)
2. Remove fill plug FIRST (side of pan, 8mm Allen)
3. Remove drain plug or pan (factory pan sealed — install aftermarket Dorman pan with drain plug)
4. Replace internal filter
5. Torque pan bolts: 70–100 in-lb (inch-pounds! — aluminum strips easily)
6. Fill with ZF 8&9 Speed ATF until trickling from fill hole at 176–203°F
7. Clear codes, test drive

### Rear Brake Service (EPB)
**⚠️ CRITICAL DIFFERENCE FROM JK:**

Rear calipers have Electric Parking Brake motors. **Cannot use C-clamp.**

1. Connect scan tool (JScan/AlfaOBD)
2. Navigate EPB module → Service Mode → **Retract EPB pistons**
3. Pistons retract electronically
4. Replace pads/rotors normally
5. Exit Service Mode → **Apply EPB**
6. Pump pedal 8–10 times

**Torque:** Rear caliper bracket 70 ft-lb, front bracket 125 ft-lb

### Spark Plug Replacement (Same as JK)
- NGK LZFR5CI-11, gap 0.043"
- Torque 13 ft-lb
- Remove upper intake manifold for Bank 2 (driver side)
- Torque intake bolts: 105 in-lb (inch-pounds!)

### Cabin Air Filter
1. Open glove box
2. Squeeze side stops inward, drop glove box
3. Slide filter tray out
4. Replace Mopar 68071668AA filter
5. Reset glove box

### Serpentine Belt
- Gates K060795 or equivalent
- 15mm wrench on tensioner
- Photograph routing first

## Known Issues

### High-Frequency Failures

| Issue | Symptoms | Fix |
|---|---|---|
| **ZF 8-speed harsh shifting** | Jerky 1-2 shift, low-speed lurching | **PCM/TCM software flash** — #1 complaint, TSB available |
| **TIPM fuel pump relay** | Pump runs with key out OR no-start/recall | External bypass harness or TIPM replacement |
| **Quadra-Lift air leak** | Corner drops, compressor runs constantly | Find leak (soap test), replace component |
| **UConnect freezes** | Screen blackout, parasitic drain | Hard reset or software update |
| **EPB failure** | "Service EPB" warning, won't release | Scan tool diagnosed, caliper replacement |
| **Sunroof drain clog** | Headliner water stains | Clear A-pillar and C-pillar drain tubes |
| **Halfshaft CV boot** | Clicking in turns | Replace halfshaft (~$150–300 parts) |
