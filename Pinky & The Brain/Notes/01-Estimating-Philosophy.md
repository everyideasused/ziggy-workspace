# Estimating Philosophy — First Principles

> *"An estimate is a hypothesis. Construction is the experiment. The estimator's job is to reduce the variance between the two."*

---

## The Estimator's Creed

A world-class estimator is not a calculator — they are a **systems thinker** who understands construction as an integrated set of interacting scopes, risks, and market conditions. The number produced is only as good as the understanding behind it.

**Three laws of estimating:**
1. **Scope completeness** beats unit cost precision every time
2. **Risk-adjusted** estimates outperform optimistic ones
3. **An estimate without assumptions documented is an opinion**

---

## Estimate Types & Accuracy Bands

| Phase | Estimate Type | Accuracy Range | Basis |
|---|---|---|---|
| Pre-design / Site Selection | ROM / Order of Magnitude | ±30–50% | Benchmarks, SF cost, analogous projects |
| Schematic Design | Conceptual | ±20–30% | Preliminary plans, assemblies |
| Design Development | Design Development | ±15–20% | 60–75% CDs, parametric quantities |
| Construction Documents | GMP / Hard Bid | ±5–10% | 100% CDs, full QTO |
| Post-Bid | Reconciled Budget | ±2–5% | Awarded contracts |

**Rule:** Always communicate accuracy bands explicitly. A $1M estimate at ±30% is a $700K–$1.3M range — both numbers matter.

## The Five Fatal Estimate Errors

### 1. Scope Gaps (Most Common)
Items not included because they weren't on the drawings — yet.
- **Prevention:** Use CSI checklist against program requirements
- **Examples:** Grease trap, fire suppression upgrade, accessibility upgrades, utility connections, temp power

### 2. Quantity Error
Wrong takeoff — usually undercount of linear footage, square footage, or count.
- **Prevention:** Independent QTO, check total vs. building SF
- **Examples:** Linear feet of storefront, count of devices, SF of flooring by type

### 3. Unit Cost Error
Using wrong unit costs for the region, market, or specification level.
- **Prevention:** Regional cost indexing, current sub pricing, recent bid data
- **Examples:** Using national average for NYC/SF/Chicago; using pre-COVID labor rates

### 4. Escalation Ignored
Pricing today for a project that bids in 6 months and builds in 12–18 months.
- **Prevention:** Apply escalation from estimate date to midpoint of construction
- **Rule of thumb:** 3–8%/year depending on market; spike years (2021–2022) hit 15–20%

### 5. Contingency Misapplication
Using contingency to "feel right" rather than quantifying risk.
- **Prevention:** Tie contingency to unknowns: design completeness, site conditions, market volatility
- **Ranges:** 5% (100% CDs, stable market) → 25% (conceptual, complex site)

## The Estimating Reasoning Chain

When asked to estimate anything, follow this sequence:

```
STEP 1: CLARIFY SCOPE
  → What is the project type? New, TI, renovation?
  → What space program? SF by use?
  → What spec level? Base building? Brand standard? Custom?
  → What's included? What's excluded? Who's doing what?

STEP 2: IDENTIFY KNOWN UNKNOWNS
  → What's NOT on the drawings yet?
  → What site conditions are assumed?
  → What code or AHJ requirements may be triggered?
  → What MEP infrastructure exists vs. needs to be new?

STEP 3: QUANTIFY
  → Run QTO or apply SF benchmarks by system
  → Cross-check: does MEP = 35–50% of TI? Does it feel right?
  → Cross-check: does total $/SF align with comparable projects?

STEP 4: PRICE
  → Apply unit costs (current, regionally adjusted)
  → Apply labor productivity for local market
  → Apply material pricing current to estimate date

STEP 5: RISK-ADJUST
  → Add contingency appropriate to phase
  → Flag high-risk line items (design TBD, conditions unknown)
  → Apply escalation from estimate date to construction midpoint

STEP 6: DOCUMENT
  → List all assumptions
  → List all exclusions
  → State accuracy band
  → Note what would change the estimate materially
```

## Scope Validation Checklist (Run on Every Estimate)

**Site / Civil**
- [ ] Site demolition and clearing
- [ ] Utility relocations (gas, electric, water, sewer, storm)
- [ ] Paving, curb, and site concrete
- [ ] Landscaping and irrigation
- [ ] Site lighting
- [ ] Signage (site + building)

**Shell (if applicable)**
- [ ] Foundation and slab
- [ ] Structure
- [ ] Exterior envelope (walls, roof, glazing)
- [ ] Roof drainage

**Tenant Improvement — Always Check**
- [ ] Demolition of existing conditions
- [ ] Slab core/cut for new plumbing
- [ ] Structural reinforcement (kitchen equipment, rooftop units)
- [ ] Storefront/entrance modifications
- [ ] ADA compliance (restrooms, path of travel, parking)
- [ ] Grease interceptor / grease trap (restaurants)
- [ ] Hood and exhaust (restaurants — often by owner but need infra)
- [ ] Make-up air unit
- [ ] Walk-in cooler/freezer (often by owner, but infra)
- [ ] Fire suppression system (new or modification)
- [ ] Fire alarm (new or modification)
- [ ] Electrical service upgrade (often underestimated)
- [ ] Data/low voltage/POS rough-in
- [ ] Drive-thru infra (speakers, canopy, menu boards, loops)
- [ ] Signage infrastructure (conduit, disconnects)
- [ ] Security/access control
- [ ] Trash enclosure
- [ ] Utility connections and fees (often $50K–$300K+ and missed)

**Soft Costs (Not Always in GC Scope)**
- [ ] Permit fees
- [ ] Utility connection fees
- [ ] Testing and inspection
- [ ] Special inspections
- [ ] Commissioning
- [ ] FF&E coordination (even if owner-furnished)

## MEP Rules of Thumb for Budget Checks

| System | % of TI Hard Cost |
|---|---|
| Plumbing | 8–15% |
| HVAC/Mechanical | 12–20% |
| Electrical | 10–18% |
| Fire Suppression | 2–5% |
| Low Voltage / Comm | 2–5% |
| **MEP Total** | **35–55%** |

**If MEP is < 30% of a TI estimate — review the scope. Something is missing or underpriced.**

## GC Markup Structure

Understanding how GCs build their number is essential to auditing it:

| Line Item | Typical Range | Notes |
|---|---|---|
| General Conditions | 8–15% of direct cost | Superintendent, PM, trailer, temp utilities |
| Overhead | 3–7% | Home office overhead allocation |
| Profit | 3–8% | Higher on smaller, riskier projects |
| Contingency (GC-held) | 1–5% | Varies by risk comfort |
| **Gross Margin** | **12–25%** | Combined overhead + profit |
| Bonds | 0.5–2% | Required on public projects, many institutional |
| Insurance | 1–2% | GL, builder's risk |

**Total loaded markup over direct cost:** 15–30% depending on project size, complexity, GC type

## Bid Leveling Discipline

Never compare "apples to apples" until you've confirmed:
1. Same scope inclusions/exclusions
2. Same alternates included
3. Same allowances for owner-furnished items
4. Same bonds and insurance structure
5. Same schedule of values format

**Red flag:** Two bids more than 15% apart almost always means different scope assumptions — not one GC being "expensive."

---

## Calibration Mindset

The best estimators maintain a running mental model of cost that updates with every project. Key calibration habits:

- **Post-bid review:** Compare estimate to awarded GMP within 2 weeks
- **Closeout review:** Compare GMP to final cost
- **Market pulse:** Check with 2–3 subs per trade quarterly
- **Historical database:** Log all completed project benchmarks by type, year, region

> *"Your estimate is only as good as your last 50 projects."*
