## Progression Scheme (Phase 1)

**Weeks 1-4:** Learn movements, establish baseline  
- Focus: Perfect form, build conditioning base  
- Reps: 8-12 per set  
- Sets: 2-3 per exercise  
- Rest: 90-120 seconds  
- Kettlebell: 6-8 kg to start  

**Weeks 5-8:** Volume accumulation  
- Add 1 set per exercise  
- Increase reps when form is perfect  
- Introduce kettlebell complexes  
- Conditioning: Add run intervals (30 sec jog / 90 sec walk)

**Weeks 9-12:** Consolidation + assessment  
- Test maxes (max reps, max weight for clean reps)  
- Prep for Phase 2 volume work  
- Conditioning: 10 min continuous easy jog
