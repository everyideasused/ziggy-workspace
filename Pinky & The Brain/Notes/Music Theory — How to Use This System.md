## 📅 Weekly Workflow

### Monday (1 hour — Theory Lecture)
1. Open [[Learn Music Theory — Comprehensive]]
2. Check which Allen lectures are scheduled this week
3. Watch/complete the lectures
4. Create learning log: **QuickAdd: Music Theory Log** (session type: Lecture)
5. Document key concepts and examples

### Wednesday (1 hour — Practice & Exercises)
1. Create learning log (session type: Practice)
2. Complete musictheory.net exercises
3. Do teoria.com drills
4. Practice in MuseScore (build chords, scales, etc.)
5. Note which concepts feel solid vs. shaky

### Friday (1 hour — Allen + MuseScore)
1. Create learning log (session type: Lecture or Practice)
2. Continue Allen videos
3. Apply concepts in MuseScore
4. Experiment with composition/transcription

### Saturday (1 hour — Listening + Analysis)
1. Create learning log (session type: Analysis)
2. Watch Yale MUSI 112 lecture (when scheduled)
3. Analyze songs using this week's concepts
4. Connect theory to music you actually listen to

### End of Week (Before Next Monday)
1. Open [[Learn Music Theory — Comprehensive]]
2. Update progress tracker
3. Check off completed tasks
4. Fill in weekly review (5 min reflection)

## 🎯 Milestone Checkpoints

**8 checkpoints built into the schedule:**

| Week | Checkpoint | What You'll Do |
|------|------------|----------------|
| 4 | Notation fluency | Sight-read treble/bass clef without hesitation |
| 8 | All key signatures | Name any major/minor key sig in under 5 seconds |
| 10 | Roman numeral analysis | Analyze diatonic progressions from lead sheets |
| 14 | Modal identification | Hear and name all 7 modes |
| 16 | Seventh chord fluency | Build and identify all seventh chord types |
| 20 | Chromatic harmony | Identify secondary dominants and borrowed chords |
| 24 | Full song analysis | Complete harmonic/melodic/formal analysis |
| 24 | Final composition | Write original piece using learned techniques |

**These are self-assessments.** Be honest. If you don't pass a checkpoint, repeat that week's material.

## 💡 Tips

1. **Connect to real music** — Always relate theory to songs you love
2. **Use both ears and eyes** — Theory + ear training = real understanding
3. **Don't rush checkpoints** — Better to master Week 8 content than fake it and struggle later
4. **Link theory to bass** — Every concept should inform your playing
5. **MuseScore is your playground** — Experiment freely
6. **Yale lectures are enrichment** — They're optional but highly valuable for context
