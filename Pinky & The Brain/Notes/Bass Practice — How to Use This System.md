# Bass Practice — How to Use This System

Quick guide for tracking your bass learning journey.

---

## 🗂️ System Overview

**Main Hub:** [[Learn Bass — Beginner to Badass]]  
**Practice Logs:** Individual notes for each session  
**Weekly Reviews:** Embedded in main project note

---

## 📅 Weekly Workflow

### Sunday (Lesson Day)
1. Open [[Learn Bass — Beginner to Badass]]
2. Check which lessons you're on this week
3. Watch/complete the lesson(s)
4. Create practice log: **QuickAdd: Bass Practice Log** (or use daily note button if added)
5. Document what you learned

### Tuesday (Practice Day)
1. Create practice log
2. Drill exercises from Sunday's lesson
3. Play along with course tracks
4. Note struggles and wins

### Thursday (Practice Day)
1. Create practice log
2. Continue drills
3. Focus on areas that felt hard on Tuesday
4. Prepare for next week's lessons (if you finish early)

### End of Week (Before Next Sunday)
1. Open [[Learn Bass — Beginner to Badass]]
2. Update progress tracker
3. Fill in weekly review (5 min reflection)
4. Check off completed lessons

## 🎯 Creating Practice Logs

**Two ways:**

### Option 1: QuickAdd Command
1. Open command palette (`Cmd+P` / `Ctrl+P`)
2. Type "QuickAdd: Bass Practice Log"
3. Hit Enter
4. Templater prompts:
   - Session type: Lesson or Practice
   - Duration: minutes

### Option 2: Daily Note Button (If Added)
1. Open today's daily note
2. Click **🎸 Bass Practice** button
3. Same Templater prompts

## 💡 Tips

1. **Be honest in logs** — Struggles are progress data, not failures
2. **Record yourself** — Even bad recordings show progress over time
3. **Don't skip reviews** — 5 min/week keeps you aware of patterns
4. **Link related notes** — If you learn a song, link it to the practice log
5. **Celebrate milestones** — Week 7 benchmark, first song mastered, etc.
