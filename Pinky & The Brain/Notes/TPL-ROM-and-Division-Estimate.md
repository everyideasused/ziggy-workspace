# TPL-ROM-Budget — ROM Estimate Template

> Copy this template for every new ROM estimate. Fill in all fields. Never leave assumptions blank.
> See [[WF-Conceptual-ROM-Budget]] for the full step-by-step process.

---

Accuracy:        ± ____% ($_______ – $_______)

[2–3 sentences describing exactly what is included]

Building / TI:
  ________ SF × $ _______ /SF                   $ ___________
  Regional multiplier applied: ___ × (_______ )

Site-specific additions:
  Drive-thru (if applicable):                    $ ___________
  Grease interceptor (if applicable):            $ ___________
  Utility connections (estimate):                $ ___________
  [Other: ________________________________]:    $ ___________

                          DIRECT COST SUBTOTAL:  $ ___________
                          Contingency (_____%):  $ ___________
                          Escalation (_____%):   $ ___________
                          (____  months to midpoint @ ____% /yr)
───────────────────────────────────────────────────────
                          HARD COST TOTAL:       $ ___________
                          Per SF:               $ _______ /SF
───────────────────────────────────────────────────────

SOFT COSTS (complete if total project budget required):
  Permits / fees:                                $ ___________
  Testing & inspections:                         $ ___________
  A/E (if applicable):                           $ ___________
  Owner PM (if applicable):                      $ ___________
  Utility connection fees (verified / est.):     $ ___________
  Other: _________________________________:      $ ___________
                          SOFT COST TOTAL:       $ ___________

═══════════════════════════════════════════════════════
                     TOTAL PROJECT BUDGET:       $ ___________
         Range (± _____%):  $ __________ – $ __________
═══════════════════════════════════════════════════════

───────────────────────────────────────────────────────
ASSUMPTIONS:
───────────────────────────────────────────────────────
1.  Shell delivery: [state what landlord/owner is providing]
2.  Regional cost index: [city], multiplier = ____
3.  Brand standard level: [standard / above-standard / custom]
4.  Drive-thru: [included / excluded / single / dual lane]
5.  FF&E: [excluded — owner-furnished]
6.  Signage: [excluded — by owner or sign contractor]
7.  Utility connection fees: [excluded / estimated at $___]
8.  Escalation: [____ months to midpoint, ____% annual rate]
9.  Contingency basis: [phase + specific risk factors]
10. [Add project-specific assumptions]

───────────────────────────────────────────────────────
EXCLUSIONS:
───────────────────────────────────────────────────────
- FF&E, furniture, owner-furnished equipment
- Signage (exterior and interior)
- Utility connection and development fees (unless noted above)
- Permit fees (unless noted above)
- A/E fees (unless noted above)
- Technology (POS, AV, security monitoring)
- Brand-standard proprietary items (unless specifically included)
- [Add project-specific exclusions]

───────────────────────────────────────────────────────
WHAT WOULD MATERIALLY CHANGE THIS ESTIMATE:
───────────────────────────────────────────────────────
INCREASES:
- [Risk 1: e.g., "Cold dark shell instead of warm — add $35–$55/SF"]
- [Risk 2: e.g., "Dual drive-thru instead of single — add $150K–$200K"]
- [Risk 3:]

DECREASES:
- [Opp 1: e.g., "Vanilla box delivery — reduce by $20–$35/SF"]
- [Opp 2:]

───────────────────────────────────────────────────────
NEXT STEPS TO IMPROVE ACCURACY:
───────────────────────────────────────────────────────
☐ Existing conditions survey ($2,000–$8,000)
☐ Utility capacity letters from each utility
☐ AHJ pre-application meeting (ADA, code compliance)
☐ Schematic design drawings (target ±15%)
☐ Geotechnical report (ground-up only)
═══════════════════════════════════════════════════════
