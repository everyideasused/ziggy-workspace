# Appendix — General Retail & National Rollout Programs

> Industry appendix for [[Construction PM Knowledge Base]] and [[Sector Profiles — Retail, Commercial, Residential, Civil]]

---

## Overview

General retail encompasses everything from apparel and electronics to home goods and specialty stores. When managed as a national rollout program (dozens to thousands of locations), the construction process becomes a logistics and standardization challenge more than a one-off project management exercise.

## Prototype-Driven Construction

### What is a Prototype?
A **prototype** is a standardized design package that represents the brand's ideal store layout, finishes, and systems. National retailers develop prototypes that are then adapted to each specific location.

### Prototype Package Contents
- Architectural prototype drawings (floor plan, RCP, elevations, sections, details)
- MEP prototype drawings (single-line diagrams, typical layouts)
- Brand standards manual (finishes, colors, materials, signage specs)
- Fixture and equipment specifications
- Prototype specifications (construction specs for typical conditions)
- Prototype budget (target cost per SF or per location)

### Site Adaptation Process
```
Prototype Package → Site Survey / As-Built Verification →
Site-Specific Adaptation (architect adjusts prototype to actual conditions) →
Landlord Review & Approval → Jurisdictional Code Review →
AHJ-Specific Amendments → Permit Submission
```

### Common Adaptation Triggers
- Column locations that conflict with prototype layout
- Existing MEP locations that don't match prototype assumptions
- Ceiling height differences
- Entry location or storefront configuration
- ADA path-of-travel requirements specific to the site
- Local code amendments that differ from prototype code basis
- Landlord design criteria (storefront design, HVAC requirements, signage restrictions)

## National Rollout Program Management

### Program Structure
```
Client (Brand/Retailer)
  └── Program Manager (PM firm or internal team)
       ├── Regional Project Managers
       │    ├── Architect(s) — may be national or regional
       │    ├── GC(s) — typically regional
       │    └── Specialty Vendors — often national (fixtures, signage, technology)
       └── Central Functions
            ├── Real Estate / Site Selection
            ├── Design / Prototype Management
            ├── Procurement / Vendor Management
            └── Finance / TIA Administration
```

### Velocity Management
For high-volume programs (50-500+ locations per year):

| Metric | Target |
|--------|--------|
| Stores in pipeline simultaneously | 20-100+ |
| Average construction duration | 6-12 weeks |
| Permit turnaround (average) | 3-6 weeks |
| Design adaptation turnaround | 2-4 weeks |
| Store openings per month | 4-20+ |

### Standardized Workflows
Every location follows the same phase-gate process:
```
1. Site Approved → 2. Lease Executed → 3. Design Adaptation →
4. Landlord Approval → 5. Permit Submission → 6. GC Bidding/Award →
7. Permit Received → 8. Construction Start → 9. Substantial Completion →
10. Fixture/Merchandise → 11. Store Opening
```

Each step has defined deliverables, responsible parties, and target durations.

## Landlord Coordination — Retail Specific

### Landlord Work vs. Tenant Work
The lease defines what the landlord provides (base building) vs. what the tenant builds:

| Typically Landlord's Work | Typically Tenant's Work |
|--------------------------|----------------------|
| Building shell (roof, exterior walls, slab) | Interior partitions and finishes |
| Base building HVAC (rooftop unit) | Distribution ductwork and controls |
| Electrical service to tenant panel | Interior electrical from panel |
| Plumbing to tenant space (stub-outs) | Interior plumbing and fixtures |
| Fire sprinkler main loop | Sprinkler head relocation/addition |
| Storefront system (sometimes) | Interior storefront elements |
| Demising walls | Interior layout |
| ADA path-of-travel to space | ADA compliance within space |

**Critical:** Verify exactly what the landlord delivers and when. Gaps between landlord's work and tenant's work are a common source of change orders and delays.

### Landlord Approval Process
```
Tenant submits design drawings to landlord →
Landlord reviews for compliance with lease criteria →
Comments issued (typically 1-2 rounds) →
Landlord approval letter issued →
Tenant can proceed to permit and construction
```
- **Timeline:** 1-4 weeks (but can be longer with difficult landlords)
- **Common issues:** Storefront design, signage, HVAC modifications, structural loads, hours of construction, insurance requirements

## Fixture & Merchandise Installation (Post-Construction)

After the GC achieves substantial completion and the CO is issued, there's a critical handoff:

### Post-Construction / Pre-Opening Sequence
```
Construction Substantial Completion → Deep Clean →
Fixture Installation (shelving, display units, checkout counters) →
Technology Installation (POS, security, digital signage, Wi-Fi) →
Merchandise Delivery and Stocking →
Visual Merchandising (product display, signage) →
Staff Training →
Soft Opening → Grand Opening
```

- **Duration:** Typically 1-3 weeks after CO
- **Coordination:** Fixture and tech vendors must be scheduled before construction ends
- **FF&E is NOT covered by TIA** — separate budget

## Technology & Low-Voltage Systems

Modern retail relies heavily on technology:

| System | Components |
|--------|-----------|
| POS (Point of Sale) | Registers, card readers, receipt printers, cash drawers |
| Network / Wi-Fi | Switches, access points, firewall, ISP circuit |
| Security | CCTV cameras, NVR, alarm panel, access control |
| Digital Signage | Displays, media players, content management |
| Music / Audio | Speakers, amplifier, streaming device |
| Inventory Management | Handheld scanners, RFID (some retailers) |
| Customer Analytics | Traffic counters, heat mapping (some retailers) |

**Coordination notes:**
- ISP circuit order should happen at lease execution (30-60+ day lead time)
- Low-voltage cabling rough-in happens during construction (before drywall close)
- Equipment installation happens post-CO (during fixture phase)
- POS integration testing is on the critical path for opening

## Signage

### Sign Types (Typical Retail)
- Channel letters (illuminated building-mounted letters)
- Monument sign (free-standing ground sign)
- Blade/projecting sign (perpendicular to building face)
- Window graphics (vinyl applied to glass)
- Interior signage (wayfinding, department identification)
- Directional/regulatory signage (parking, ADA, hours)

### Sign Permitting
- **Landlord approval** required before fabrication (sign criteria in lease)
- **City sign permit** required (size, height, illumination restrictions)
- **HOA/design review** may apply in some shopping centers
- **Lead time:** Design → landlord approval → permit → fabrication → installation = 6-12 weeks total
- **Start sign process at design adaptation phase** to avoid being on critical path

## Common Risks — Retail Rollout

| Risk | Mitigation |
|------|------------|
| Landlord base building not ready | Track landlord deliverables; start early communication |
| Permit delays in unfamiliar jurisdiction | Use the [[AHJ Research Methodology]] framework; budget extra time |
| Existing conditions differ from as-builts | Always field-verify before design; budget field condition contingency |
| GC quality inconsistency across regions | Pre-qualify GC bench; standardize scope and specs; regular QC audits |
| ISP circuit not ready for opening | Order at lease execution; follow up weekly; have backup plan (cellular) |
| Prototype doesn't fit the space | Invest in quality site-specific adaptation; don't force the prototype |
| TIA overage on multiple locations | Track program-level TIA performance; identify cost drivers early |
