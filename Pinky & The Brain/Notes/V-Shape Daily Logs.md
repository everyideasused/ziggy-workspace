# V-Shape Program — Daily Workout Logs

> **How to use:** Copy the appropriate template for your workout day. Fill in the numbers. Tag with the date. Ziggy can parse these structured logs to track trends, flag stalls, and recommend progressions.
>
> **File per session:** Create a new note for each workout: `YYYY-MM-DD — Day X.md` and paste the matching template below.
